### 实验要求：<br>

开发一个Java命令行程序，实现多项式的输入，求值，求导。在程序运行之后，用户在命令行输入表达式或命令后，程序反馈结果。<br>

### 数学模型：<br>

>输入数据：有效的多项式，有效的命令。<br>

>输出数据：表达正确的多项式，在相关命令下得出的正确结果。<br>

>优化目标：多项式可以使用减法，可以使用“^”代表幂运算。<br>

>约束条件：多项式不包含其他非法字符(如：括号，小数点)。<br>

具体要求请见`“Lab1实验手册.pdf”`。
